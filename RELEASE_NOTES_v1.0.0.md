# SKAT Motor Tester v1.0.0

Первый публичный релиз Android-приложения для стенда **SKAT-Tenzo**.

## Установка

1. Скачайте файл **SKAT-Motor-Tester-1.0.0.apk** из раздела Assets.
2. Разрешите установку из неизвестных источников.
3. Установите приложение на устройство с Android 8.0+ и BLE.

## Системные требования

- Android 8.0 (API 26) и выше
- Bluetooth Low Energy (BLE)
- Разрешения: Bluetooth; на Android 11 и ниже — геолокация при сканировании

## Подключение

1. Включите стенд SKAT-Tenzo.
2. В приложении: **«Найти устройство»** → выберите **SKAT-Tenzo**.
3. Разрешите доступ к Bluetooth (и геолокации при запросе системы).

## Использование

- **Главный экран** — сила тяги, ток, напряжение, краткие графики.
- **Arm / Disarm** — разрешение управления приводом.
- **Троттл** — мощность ESC (активен после Arm).
- **Настройки (шестерёнка)** — калибровка тензодатчика и датчика тока.
- **Графики** — подробный просмотр, полноэкранный режим, экспорт CSV/PNG.

## Калибровка (кратко)

**Масса:** снять нагрузку → Обнулить → эталонная масса → Калибровать по массе.

**Ток/напряжение:** указать шунт → Применить → Перекалибровать INA226; напряжение — по мультиметру; ток — Обнулить ток при Disarm.

Подробная инструкция — в [README.md](https://github.com/reduct0r/SKAT-Load-cell/blob/master/README.md).

## Состав релиза

- BLE-телеметрия и управление PWM
- Экран калибровки HX711 и INA226
- Графики с экспортом
- Сохранение положения троттла при навигации

Прошивка ESP32 в этот релиз не входит.
